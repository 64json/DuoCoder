## Boolean Type Variables

1. True values
    * `True`
    * `-1`
    * `1`
    * `2`
    * `'any_string'`
    * \+ more
    
2. False values
    * `False`
    * `0`
    * `''`
    * `None`
    * \+ more

3. `bool()` function returns either `True` or `False`.

### What it looks like?
`bool (name) = (True or False);`
