## If / Else Statement

1. If statement
    ```
    if condition:
        statement
    ```

2. Else If statement
    ```
    elif condition:
        statement
    ```

3. Else statement
    ```
    else:
        statement
    ```
    
4. _If_ statement should precede before _Else If_ and _Else_ statement.

5. _Else If_ statement cannot come after _Else_ statement.

6. _Pass_ statement does nothing but acts as a placeholder.
